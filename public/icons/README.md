# App icons

Положи сюда PNG-иконки приложения (PWA + iOS). Все файлы — квадратные PNG.

| Файл                   | Размер    | Назначение                                            |
| ---------------------- | --------- | ----------------------------------------------------- |
| `icon-192.png`         | 192×192   | PWA, иконка обычного вида (`purpose: any`)             |
| `icon-512.png`         | 512×512   | PWA, крупная иконка / сплэш (`purpose: any`)           |
| `maskable-512.png`     | 512×512   | Android adaptive icon (`purpose: maskable`)            |
| `apple-touch-icon.png` | 180×180   | Иконка на домашнем экране iOS                          |

Подсказки:

- **maskable** — логотип должен помещаться в «безопасную зону»: центральные ~80 % (отступ ~10 % по краям), фон заливай сплошным цветом (`#0a0a0a` или `#0ea5e9`), иначе на Android углы обрежет.
- **apple-touch-icon** — без прозрачности (iOS сам скруглит углы), фон сплошной.
- Фавикон браузера остаётся отдельно: `src/app/favicon.ico`.
- После добавления файлов ничего пересобирать не нужно — пути уже прописаны в `public/manifest.json` и `src/app/layout.tsx`.
